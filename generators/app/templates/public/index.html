Webpack riot js


Description

This generator webpack setup adds babel for ES2015 transformation, autoprefixer for autoprefixing css, and Modernizr for feature detection. And of course riotjs.

Sample code for modernizr with a polyfill for placeholder where the browser is missing that feature.

if (!Modernizr.input.placeholder) {
require('./js/polyfills/placeholder');
} 

        app
          |-js
          |-styles
          |-tags
               |-elements
               |-functional
               |-pages

Create new element on command line with:

yo riot-element new-tag-name [--path sub/path/to/store/my/tag]

Stored under app/tags/elements/tagname or app/tags/elements/sub/path/to/store/my/tag/tagname

Create new page element on command line with:

yo riot-element:page new-tag-name [--path sub/path/to/store/my/tag]

Stored under app/tags/pages/tagname or app/tags/pages/sub/path/to/store/my/tag/tagname

Create new functional element on command line with:

yo riot-element:functional new-tag-name [--path sub/path/to/store/my/tag]

Stored under app/tags/functional/tagname or app/tags/functional/sub/path/to/store/my/tag/tagname
